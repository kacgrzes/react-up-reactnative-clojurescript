## Idea of Clojure

- clean syntax / almost no syntax
- only immutable data structures
- embraces JVM platfrom
- data oriented (code-as-data)
- macro system - takes code and returns new code

---

## Code as data

``` clojure
(+ 1 1) ; 2
'(+ 1 1 ) ; a list of a function and two numbers 
```
