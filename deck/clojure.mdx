## Idea of Clojure

- clean syntax / almost no syntax
- only immutable data structures
- embraces JVM platfrom
- data oriented (code-as-data)
- macro system - takes code and returns new code

```notes
macro system uses code-as-data to extend language syntax

With macros, you can extend Clojure to suit your problem
space, building up the language
```
